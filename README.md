# 🌐 My Portfolio

Welcome to my personal portfolio website. This project showcases my skills, projects, and work experience as a software developer. It includes web and mobile projects, personal projects, and freelance work.

🔗 **Live website:** [mdshamsurrahmansami.netlify.app](https://mdshamsurrahmansami.netlify.app/)

## 🐅 Features

- Responsive and modern UI built with **React** and **Tailwind CSS**
- Animations with **Framer Motion**
- Dark mode support
- Sections include:
  - About Me
  - Work Experience
  - Skills
  - Contact Form
- Contact form integrates with **FormSubmit** for email submissions
- Social links to GitHub and LinkedIn
- Fully deployable on **Netlify**

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Sami-115667/My-Portfolio.git
```

### 2. Navigate into project directory

```bash
cd My-Portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run development server

```bash
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```

## ☁️ Deployment

This portfolio is hosted on **Netlify**. To redeploy:

1. Push changes to your GitHub repository
2. Connect the repository to Netlify
3. Trigger a manual deploy or enable automatic deploy on push
